Architettura & Business Model

Stack tecnologico, API consigliate, modello di revenue e schema database per il MVP di WanderQuest.

📱
Frontend
React Native per mobile (iOS + Android) con Expo. Next.js per la versione web PWA. State management con Zustand, data fetching con React Query. Mappe con Mapbox GL JS.
React Native Next.js Mapbox Zustand TailwindCSS
⚡
Backend
Node.js + Express (API REST) con PostgreSQL + PostGIS per dati geospaziali. Redis per cache e contatori voti real-time. RabbitMQ per code asincrone (moderazione foto, premi).
Node.js PostgreSQL PostGIS Redis RabbitMQ
☁️
Infrastruttura
Supabase come BaaS (auth, database, storage). Cloudflare R2 per foto (costo zero egress). Nginx come API Gateway con rate limiting. Firebase FCM per notifiche push.
Supabase Cloudflare R2 Firebase Nginx
✈️
API Voli
AeroDataBox (consigliata): real-time status, logo carrier, storico voli. 500 richieste gratuite/mese. Fallback su OpenSky Network per dati di posizione illimitati.
AeroDataBox AviationStack OpenSky
🌤️
API Meteo & Mappe
OpenWeatherMap per meteo corrente (1k req/giorno free). Open-Meteo per storico climatico illimitato e gratuito. Google Places API per ristoranti e POI locali.
OpenWeatherMap Open-Meteo Google Places
💳
Pagamenti & AI
Stripe per abbonamenti ricorrenti e payout ai vincitori. KYC con Stripe Identity. OpenAI GPT-4o per itinerari e moderazione. AWS Rekognition per analisi foto.
Stripe OpenAI GPT-4o AWS Rekognition
